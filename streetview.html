NOTE:

Ce code provient d'un exemple fourni par Google.

La partie YOUR_API_KEY doit être remplacée par un code personnel. C'est une méthode utilisée par Google pour contrôler l'accès à ses services et prévenir des usages abusifs (trop grand nombre de connexions).

Voici l'endroit où vous pouvez créer une clé API pour Streetview: https://console.developers.google.com/

  • Aller sous API Manager > Credentials.
  • Vous devrez commencer par créer un "Project".
  • Ensuite vous pourrez créer un "API Key".
  • Une fois créé, il faut activer les API souhaitées.
  • Pour cet exemple, c'est la "Google Maps Embed API".
